# EquityForm — Legal Intake Frontend

This is a frontend-only React + TypeScript app that implements a multi-step intake questionnaire for populating DOCX templates (Operating Agreement and Subscription Agreement). It uses Zustand for local state with localStorage persistence, React Hook Form + Zod for validation, and a modular component architecture.

Run locally:

1. npm install
2. npm run dev

This is an initial scaffold implementing the core flows, mock data, placeholder mapping utility, and a review page. Next steps: polish validations, conditional logic, compute derived fields, and add export integration.
